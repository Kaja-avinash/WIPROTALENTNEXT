## 📌 Overview

This assignment demonstrates thread creation using the **Runnable** interface. A thread repeatedly generates and displays random colours selected from a predefined array.

---

## 🎯 Objective

- Implement Runnable Interface
- Create a Thread object
- Generate random colours
- Display colours repeatedly
