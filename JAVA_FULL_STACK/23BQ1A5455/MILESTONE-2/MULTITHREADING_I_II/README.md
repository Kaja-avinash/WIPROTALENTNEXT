# рЯУЦ About Repository

This repository contains a structured collection of **Java Multithreading** programs developed using **Eclipse IDE**.

The purpose of this repository is to understand how multiple threads execute concurrently, how CPU scheduling works, and how Java provides APIs for creating and controlling threads.

The repository progresses from basic thread creation to advanced thread control mechanisms through hands-on examples and a mini project.

# рЯОѓ Objectives

- Learn Java Multithreading concepts.
- Create multiple threads.
- Understand the Thread lifecycle.
- Execute concurrent programs.
- Explore Thread priorities.
- Learn Thread control mechanisms.
- Understand `Thread.sleep()`.
- Develop practical multithreading applications.
