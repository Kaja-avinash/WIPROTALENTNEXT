## 📌 Overview

This assignment demonstrates assigning different priorities to multiple threads using Java's predefined priority constants.

---

## 🎯 Objective

- Create three threads
- Assign MAX, NORM and MIN priorities
- Execute threads concurrently
- Observe thread scheduling
