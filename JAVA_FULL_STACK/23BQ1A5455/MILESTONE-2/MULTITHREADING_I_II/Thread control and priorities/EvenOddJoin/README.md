## 📌 Overview

This assignment demonstrates synchronization between two threads using the `join()` method. The even-number thread completes before the odd-number thread starts.

---

## 🎯 Objective

- Create two threads
- Print even numbers
- Print odd numbers
- Synchronize execution using `join()`
