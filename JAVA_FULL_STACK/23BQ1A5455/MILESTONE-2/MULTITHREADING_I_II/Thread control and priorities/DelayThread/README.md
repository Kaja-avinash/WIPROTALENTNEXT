## 📌 Overview

This assignment demonstrates thread control using the `sleep()` method. The thread prints numbers from **1 to 10** and pauses for **5 seconds** after printing **5**.

---

## 🎯 Objective

- Create a thread
- Print numbers from 1–10
- Pause execution using `Thread.sleep()`
- Resume execution after the delay
