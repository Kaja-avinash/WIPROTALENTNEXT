# 📌 Overview

This repository contains hands-on assignments demonstrating the use of **JUnit 4 assertion methods** in Java using **Eclipse IDE**. These assignments focus on writing unit tests to validate array operations and boolean conditions using commonly used JUnit assertions.

---

# 🎯 Learning Objectives

- Understand JUnit Assertions
- Write Unit Test Cases
- Validate Arrays
- Test Boolean Conditions
- Execute Automated Tests
- Learn Eclipse JUnit Integration

# 🧪 JUnit Assertions Covered

| Assertion | Description |
|-----------|-------------|
| `assertArrayEquals()` | Verifies that two arrays contain the same elements in the same order. |
| `assertTrue()` | Verifies that a condition evaluates to `true`. |
| `assertFalse()` | Verifies that a condition evaluates to `false`. |

# ▶️ How to Run

1. Open the project in Eclipse IDE.
2. Ensure **JUnit 4 Library** is added.
3. Open the required test class.
4. Right-click the file.
5. Select:

```text
Run As → JUnit Test
```
