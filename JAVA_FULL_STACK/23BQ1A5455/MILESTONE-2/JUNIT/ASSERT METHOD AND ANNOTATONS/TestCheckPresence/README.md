## 📌 Overview

This assignment demonstrates how to verify boolean results using **JUnit 4**. The `checkPresence()` method is tested with both matching and non-matching strings using `assertTrue()` and `assertFalse()`.

---

## 🎯 Objective

- Learn `assertTrue()`
- Learn `assertFalse()`
- Test boolean methods
- Execute JUnit test cases
