## 📌 Overview

This project demonstrates the fundamentals of **JUnit 4 Unit Testing** in Java using **Eclipse IDE**. It focuses on testing a string concatenation method using the `assertEquals()` assertion.

---

## 🎯 Objective

- Learn JUnit 4 basics
- Create and execute test cases
- Validate expected and actual results
- Understand automated unit testing

---

## 🛠️ Tech Stack

- ☕ Java
- 🧪 JUnit 4
- 💻 Eclipse IDE

## ▶️ Run

1. Open the project in Eclipse.
2. Right-click **TestStringConcat.java**.
3. Select **Run As → JUnit Test**.
