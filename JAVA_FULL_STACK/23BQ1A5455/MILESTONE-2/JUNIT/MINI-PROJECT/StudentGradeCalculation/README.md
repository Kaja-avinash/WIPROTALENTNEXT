# 📌 Overview

This repository contains the **JUnit Mini Project** developed as part of the **Wipro Java Full Stack Program**. The project demonstrates **Student Grade Calculation**, **Custom Exception Handling**, and **JUnit 4 Unit Testing** by implementing a complete set of **9 test cases** to validate different scenarios.

---

# 🎯 Objectives

- Calculate student grades based on average marks
- Validate student information
- Handle invalid inputs using custom exceptions
- Perform unit testing using JUnit 4
- Verify business logic through automated test cases

---

# 🛠️ Tech Stack

- ☕ Java
- 🧪 JUnit 4
- 💻 Eclipse IDE

# 📊 Grade Evaluation Logic

| Average Marks | Grade |
|--------------:|:-----:|
| 80 and above | A |
| 60 – 79 | D |
| Below 60 | F |

# ▶️ Execution Steps

1. Open the project in Eclipse.
2. Ensure **JUnit 4** is added to the build path.
3. Navigate to:

```text
com.mile1.test
```

4. Right-click:

```text
StudentGradeTest.java
```

5. Select:

```text
Run As → JUnit Test
```
