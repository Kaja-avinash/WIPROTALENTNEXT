## 📌 Overview

This assignment demonstrates how to create and execute a **JUnit 4 Test Suite**. Instead of running individual test classes, the suite executes multiple test cases together using `@RunWith(Suite.class)` and `@Suite.SuiteClasses`.

---

## 🎯 Objective

- Create a JUnit Test Suite
- Group multiple test classes
- Execute all tests with a single run
- Understand test suite annotations
