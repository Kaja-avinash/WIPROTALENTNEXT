This repository contains all hands-on assignments and the mini project completed as part of the **JUnit Module** in the **Wipro Java Full Stack Program**.

The module focuses on **Java Unit Testing**, **Assertions**, **JUnit Annotations**, **Test Suites**, **Exception Testing**, and **Test-Driven Development (TDD)** using **JUnit 4**.
