# 📖 Overview

This repository contains the **Day 8** Java programming exercises completed as part of the **Logic Building Hour Plan**.

The primary objective of Day 8 is to strengthen programming skills through **digit manipulation**, **conditional logic**, and **iterative processing**. The exercises focus on extracting digits from numbers and performing various arithmetic operations based on user-defined conditions.

---

# 🎯 Learning Objectives

By completing these exercises, I learned to:

- ✔ Extract digits from an integer efficiently
- ✔ Calculate the sum of all digits
- ✔ Compute the sum of even digits
- ✔ Compute the sum of odd digits
- ✔ Build menu-driven Java applications
- ✔ Apply conditional logic with loops
