# 📖 Overview

This repository contains the **Day 2** Java logic-building exercises completed as part of the **Logic Building Hour Plan**.

The primary objective of Day 2 is to build a strong foundation in Java by understanding:

- Conditional Statements
- Looping Constructs
- Decision Making
- Number-based Logic
- Iterative Programming
