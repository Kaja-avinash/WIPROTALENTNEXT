# 📖 Overview

This repository contains the **Day 6** Java programming exercises completed as part of the **Logic Building Hour Plan**.

The focus of Day 6 is to develop mathematical problem-solving skills by implementing algorithms involving **prime numbers**, **factorials**, and the **Fibonacci sequence** using loops, conditional statements, and iterative logic.
