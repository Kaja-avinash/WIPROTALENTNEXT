# 📖 Overview

This repository contains the **Day 9** Java logic-building exercises completed as part of the **Logic Building Hour Plan**.

The focus of Day 9 is on solving **palindrome-based problems**, strengthening understanding of digit manipulation, frequency analysis, and algorithmic thinking through practical Java implementations.

---

# 🎯 Learning Objectives

By completing these exercises, I learned to:

- ✔ Check whether a number is a palindrome
- ✔ Reverse integers using mathematical operations
- ✔ Analyze digit frequencies
- ✔ Determine whether a palindrome can be formed by rearranging digits
- ✔ Apply loops and conditional logic to solve algorithmic problems
