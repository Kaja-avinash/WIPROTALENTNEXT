# 📖 Overview

This repository contains the **Day 10** Java programming exercises completed as part of the **Logic Building Hour Plan**.

Day 10 focuses on solving both **string-based** and **number-based** programming problems using Java. The exercises emphasize string manipulation, palindrome verification, frequency analysis, and algorithmic problem-solving techniques.

---

# 🎯 Learning Objectives

By completing these exercises, I learned to:

- ✔ Process and manipulate strings efficiently
- ✔ Convert words into uppercase
- ✔ Check whether a string is a palindrome
- ✔ Analyze digit frequencies
- ✔ Strengthen logical reasoning using strings and numbers
- ✔ Apply Java's built-in string methods effectively
