# 📖 Overview

This repository contains the **Day 5** Java logic-building exercises completed as part of the **Logic Building Hour Plan**.

The objective of Day 5 is to strengthen problem-solving skills by applying **division**, **modulus operations**, and **looping constructs** to solve number-based programming challenges.
