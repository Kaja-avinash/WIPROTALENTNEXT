# 📖 Overview

This repository contains the **Day 3** Java logic-building exercises completed as part of the **Logic Building Hour Plan**.

The primary objective of Day 3 is to strengthen problem-solving skills by using **functions (methods)**, **division (`/`)**, and **modulus (`%`)** operations to solve number-based logic problems.

---

# 🎯 Learning Objectives

After completing these exercises, I gained practical experience in:

- ✔ Creating and using Java methods
- ✔ Returning values from methods
- ✔ Applying division and modulus operators
- ✔ Extracting digits from numbers
- ✔ Working with positive and negative numbers
- ✔ Building reusable logic
