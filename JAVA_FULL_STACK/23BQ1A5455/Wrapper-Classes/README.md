# 📖 About Repository

This repository contains all assignments and learning material related to **Wrapper Classes** from the Wipro Java Full Stack curriculum.

Wrapper Classes bridge the gap between Java's primitive data types and objects, enabling primitives to work seamlessly with Collections Framework, Generics, APIs, databases, and modern Java libraries.

This repository provides both conceptual understanding and practical implementation through real coding exercises.
