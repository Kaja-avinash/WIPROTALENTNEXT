# 📖 About Repository

This repository contains all assignments, notes, and mini projects from the **Abstraction / Packages / Exception Handling** module of the Wipro Java Full Stack curriculum.

The objective of this module is to strengthen object-oriented programming concepts while introducing advanced Java features such as abstract classes, interfaces, lambda expressions, package organization, and robust exception handling.

Unlike beginner Java programs, this module emphasizes **software design, modularity, code reusability, and defensive programming**.

---

# 🎯 Module Overview

This module focuses on six major Java concepts:

- Abstract Classes
- Final Keyword
- Packages
- Interfaces
- Java 8 Interface Enhancements
- Exception Handling

The module concludes with a **Student Grade Calculation System**, a mini project that integrates arrays of objects, custom exceptions, validation logic, and package-based application design.
