# 📖 About Repository

Java Input/Output (I/O) Streams enable applications to communicate with external resources such as files, devices, networks, and memory.

This repository covers the complete **I/O Streams** module from the Wipro Java Full Stack curriculum.

It introduces Java's file handling mechanisms, stream hierarchy, serialization techniques, and concludes with a practical **Employee Management System** that stores employee records using files.

---

# 🎯 Module Overview

The repository covers the following topics:

- Introduction to I/O
- I/O Operations
- Object Serialization
- Employee Management System (Mini Project)

The assignments gradually progress from reading and writing files to object persistence using serialization.
